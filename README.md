# ConverteX - Document Format Converter

ConverteX is a web-based document conversion tool built with Python and Streamlit. It allows users to convert documents between PDF, DOCX, and TXT formats easily through a simple and responsive interface.

## Features

- Upload files in '.pdf', '.docx', or '.txt' format
- Convert between the following formats:
  - TXT -> PDF
  - DOCX ->  PDF
  - PDF ->TXT
  - PDF -> DOCX
- Download converted files instantly
- Light and dark mode toggle
- Animated background with responsive UI effects

## Technologies Used

- Frontend/UI: Streamlit + custom HTML/CSS
- Backend: Python
- Libraries:
  - Python StreamLit
  - fpdf – Create PDF files
  - python-docx – Read DOCX files
  - pdf2docx – Convert PDF to DOCX
  - fitz (PyMuPDF) – Extract text from PDFs
